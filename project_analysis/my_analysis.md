Project Overview

WOW Cafe is a modern café website designed to provide users with a smooth and premium coffee-shop experience online. The website will showcase the café menu, locations, best-selling items, customer reviews, company story, and direct WhatsApp communication.

The design should focus on:

Clean UI
Mobile-first responsiveness
Smooth user experience
Premium café branding
Modern animations and interactions
